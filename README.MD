##Oxidos CMS
Бесплатный мультиязычный движок предназначен для создания корпоративных сайтов, новостных блогов, сайтов с большим информационным контентом и также подойдет для создания сайтов-визиток. Несмотря на перечисленные виды сайтов он имеет большое количество настроек, которые позволяют использовать его практически для любых целей.

##Как установить
Пожалуйста, ознакомьтесь с инструкциями по установке, включенными в дистрибутив.

##Особенности
* Быстрая скорость работы скрипта
* Возможность создавать категории неограниченной вложенности
* Возможность добавлять различный контент: статьи, статические страницы, новости, баннеры, фото
* Возможность создавать высоконагруженный сайты
* Возможность добавлять информацию на разных языках
* Простой и интуитивно понятный интерфейс
* Админка написана на фреймворке Bootstrap 3 (поддержка разрешений мобильных устройств)
* CMS написана на фреймворке Kohana 3.3
